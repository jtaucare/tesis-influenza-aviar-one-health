# Sistema de Vigilancia y Alerta Temprana para Influenza Aviar bajo el Enfoque Una Salud

## Descripción

Este repositorio contiene la documentación, el código, los datos procesados y los productos derivados de mi proyecto de tesis doctoral, cuyo objetivo es diseñar un marco metodológico para la implementación de un sistema de vigilancia y alerta temprana de influenza aviar bajo el enfoque **Una Salud (One Health)**.

El proyecto integra metodologías de inferencia causal, Diagramas Acíclicos Dirigidos (DAG), vigilancia basada en indicadores (IBS) y vigilancia basada en eventos (EBS), inteligencia artificial y visión por computadora para fortalecer la detección temprana de brotes y apoyar la toma de decisiones en salud animal, humana y ambiental.

Además, este repositorio sigue los principios de **Ciencia Abierta**, promoviendo la organización, documentación, reproducibilidad y trazabilidad de todo el proceso de investigación.

---

## Objetivos

### Objetivo general

Diseñar un marco metodológico para la implementación de un sistema de vigilancia y alerta temprana de influenza aviar bajo el enfoque **Una Salud (One Health)**, integrando inferencia causal, vigilancia basada en indicadores (IBS), vigilancia basada en eventos (EBS), inteligencia artificial y visión por computadora para fortalecer la detección temprana y la toma de decisiones epidemiológicas.

### Objetivos específicos

* Construir un modelo causal basado en **Diagramas Acíclicos Dirigidos (DAG)** que represente las relaciones entre las variables del enfoque Una Salud y la detección temprana de influenza aviar.
* Diseñar un marco metodológico para integrar fuentes de información estructuradas y no estructuradas mediante mecanismos de vigilancia **IBS** y **EBS**.
* Aplicar el marco metodológico a un caso de estudio de influenza aviar en Chile para evaluar el impacto de tecnologías de monitoreo automatizado en los tiempos de detección y en el fortalecimiento de la respuesta epidemiológica.
